单因子策略易于理解，但在某些市场机制下常表现不佳。组合多个因子可产生更稳健结果。AI 可动态加权因子并检测简单模型忽略的交互。

本文讲解如何为选股构建 AI 多因子排名系统。

## 经典因子

研究最充分的股票因子包括：

- **价值**：相对基本面便宜的股票
- **动量**：近期表现强的股票
- **质量**：盈利稳定、负债低的公司
- **低波动**：价格波动较小的股票
- **规模**：小公司历史上跑赢

每个因子在某些时期有效，另一些时期失效。

## 构建多因子分数

简单方法是给每个因子赋 z-score 再组合。

```python
import pandas as pd
from scipy.stats import zscore

factors = pd.DataFrame({
    'value': df['pe_ratio'].rank(ascending=True),
    'momentum': df['return_6m'].rank(ascending=False),
    'quality': df['roe'].rank(ascending=False),
    'low_vol': df['volatility'].rank(ascending=True)
})

factors_z = factors.apply(zscore)
df['combined_score'] = factors_z.mean(axis=1)
```

这给了每只股票一个综合多个特征的单一分数。

## AI 增强

机器学习可通过以下方式改进多因子排名：

- 从历史数据学习最优因子权重
- 检测因子间非线性交互
- 随机制变化调整权重
- 加入情绪和期权流等另类特征

梯度提升模型可从因子值预测未来收益：

```python
from lightgbm import LGBMRegressor

model = LGBMRegressor()
model.fit(X_train[factors.columns], y_train)
```

## 机制依赖的因子权重

某些因子在特定环境表现更好：

- 动量在趋势市场有效
- 价值在复苏期有效
- 质量在下行期有效
- 低波动在不确定市场有效

机制分类器可动态调整因子权重。

## 风险考虑

- **因子拥挤**：过多资本追逐相同因子降低收益
- **换手**：再平衡过频增加成本
- **数据挖掘**：过去因子表现可能不重复
- **集中**：排名靠前股票可能聚在一个行业

用行业中性、换手限制和样本外验证来缓解。

## 实用工作流

1. 选择流动性好的股票池
2. 收集因子数据
3. 处理缺失值和异常值
4. 标准化并组合因子
5. 训练模型或使用规则权重
6. 含真实成本的回测
7. 每月或每季度再平衡

## 总结

多因子排名是 AI 在股票交易中最实用的应用之一。通过组合互补信号并适应市场条件，你可构建比任何单一因子更稳健的选股流程。
