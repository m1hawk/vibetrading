## 一、为什么需要 v4？

v3 阶段已经超额完成「100 篇内容上线」的原始目标，实际产出 136 篇。但回到业务目标本身，我们发现：

1. **数量已经够多，索引才刚开始**：大量页面尚未被 Google 抓取和索引，价值未释放。
2. **用户明确要求不做 app/产品对比**：需要把内容方向重新校准到开源项目、策略、Reddit/YouTube 转化。
3. **YouTube 内容转化仍是缺口**：虽然 v3 补齐了 5 个 YouTube 主题，但 YouTube 生态变化快，仍可持续更新。
4. **GSC 后台已有部分页面被索引**：说明域名已被 Google 关注，现在是集中推送的好时机。
5. **关键词报告显示低 KD 机会词还有很多**：`ai trading platform` (KD 19)、`best ai trading app` (KD 11)、`ai investing` (KD 19) 等高流量低难度词尚未被充分利用。

因此 v4 的核心是：**索引优先 → 质量加固 → 流量验证**，不再盲目新增主题，而是让已有内容被看见、被点击。

## 五、内容质量标准（每篇必检）

### Frontmatter 模板
```yaml
---
title: "Your Post Title"
description: "150-160 character meta description with target keyword."
date: "2026-07-16"
category: "Guides" # Guides | Tutorials | Strategies | Reviews | Tools
tags: ["ai trading", "open source", "github"]
readTime: "10 min read"
featured: false
lang: "en" # or "zh"
faq:
  - question: "Question 1?"
    answer: "Answer 1."
  - question: "Question 2?"
    answer: "Answer 2."
---
```

### 发布前检查清单
- [ ] 英文 ≥ 800 字；教程/对比类英文 ≥ 1500 字；中文 ≥ 800 字
- [ ] 标题、首段、至少 1 个 H2、Meta description 中包含主关键词
- [ ] 至少 3 个站内内链（指向 `/blog/[slug]`）
- [ ] 至少 1 个代码块 / 对比表格 / 图片
- [ ] 包含 `<Disclaimer />`
- [ ] 包含 FAQ frontmatter（自动生成 FAQPage Schema）
- [ ] 中文版本本地化：案例、术语、监管提示
- [ ] 文末「相关阅读」推荐 2-3 篇

### 不做清单
- [ ] **不写付费 app/产品对比**（如 TrendSpider、3Commas、Kavout、eToro 等商业工具深度评测）
- [ ] 不写保证收益、夸大效果的表述
- [ ] 不堆砌无意义关键词
- [ ] 不直接搬运 YouTube 视频，只转化策略为可回测代码 + 风险分析

## 六、内容来源与转化策略

### GitHub 转化（保持，月度刷新）
- 当前已覆盖 25 个 GitHub 相关主题，足以支撑「开源 AI Trading」权威定位。
- **允许方向**：开源项目对比（如 `freqtrade-vs-hummingbot-crypto`、`tradingagents-vs-ai-hedge-fund`）、项目安装教程、策略代码 walkthrough。
- **不允许方向**：付费 SaaS 工具评测、未开源的 app 推荐。
- 后续每月更新 star 数和功能描述，用 GitHub API 自动校准。

### Reddit 转化（保持，热点跟进）
- 已覆盖过拟合、回测偏差、胜率 vs 期望值、Python 栈、数据成本、散户 vs 机构、骗局、模拟转实盘等核心痛点。
- **允许方向**：把 r/algotrading 高赞讨论转化为「问题 + 代码 + 风险」文章。
- **不允许方向**：推荐具体付费服务或 signal 群组。

### YouTube 转化（5 个主题已补齐，持续更新）
- 不直接搬运视频，而是把高播放视频中的策略转化为 **可回测代码 + 风险提示**。
- 每篇标注参考的视频来源（不嵌入视频，仅文字引用以降低版权风险）。
- 目标关键词：`ai trading youtube`、`convert youtube strategy to backtest`、`fake ai trading youtube`、`best ai trading youtube channels`。
- 每月扫描一次 YouTube 热门，补充 1-2 个新主题。
