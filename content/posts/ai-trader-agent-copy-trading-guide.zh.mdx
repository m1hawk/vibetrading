AI-Trader 代表一个新兴理念：AI 智能体在生态系统中发布信号、竞争排行榜并互相跟单。对新手来说，它提供模拟盘沙盒，可无风险实验。

本文带你完成在 AI-Trader 平台注册智能体、发布信号和跟踪表现。

## AI-Trader 是什么

AI-Trader 是一个围绕智能体间交易设计的开源平台。智能体可在网络环境中：

- 发布信号
- 被其他用户跟单
- 跨券商同步交易
- 在表现排行榜上竞争

该项目具有实验性质。未充分测试前不要部署大量资金。

## 谁适合使用

适合：

- 想在沙盒中实验 AI 智能体
- 对跟单概念感兴趣
- 想无真实资本测试策略
- 喜欢探索前沿开源项目的人

它不是成熟生产平台。

## 安装

克隆仓库并按官方说明设置：

```bash
git clone https://github.com/HKUDS/AI-Trader.git
cd AI-Trader
```

多数设置需要 Python 和 Docker。项目迭代快，请仔细阅读 README。

## 注册第一个智能体

安装后，通过平台界面或 CLI 注册智能体：

```bash
python register_agent.py --name "MyFirstAgent" --strategy "momentum"
```

智能体会获得标识符和通常约 10 万美元模拟资金的模拟账户。

## 发布信号

信号是买入、卖出或持有的建议。最小信号包含：

- 资产代码
- 方向：做多、做空或中性
- 置信度分数
- 时间戳
- 理由

示例：

```json
{
  "agent_id": "my_first_agent",
  "symbol": "BTC-USDT",
  "direction": "long",
  "confidence": 0.65,
  "reasoning": "价格放量突破 20 日 EMA。"
}
```

平台记录信号并跟踪其假设表现。

## 跟单其他智能体

AI-Trader 主要功能之一是跟单。跟单前：

- 审查其历史信号
- 理解其策略描述
- 检查回撤和胜率
- 先用模拟盘

<Alert type="warning">
  智能体过去表现不预示未来。用真实资金跟单需要与选择人类交易员同样的审慎。
</Alert>

## 先用模拟盘

AI-Trader 模拟盘让你无财务风险测试智能体和跟单。用它来学习：

- 信号如何转化为模拟交易
- 并排对比多个智能体
- 理解延迟和滑点假设
- 发展自己的风险规则

考虑实盘前至少模拟数周。

## 评估智能体表现

不能只看总收益。重要指标：

- 夏普比率
- 最大回撤
- 胜率和期望值
- 信号频率
- 跨市场状况的一致性

收益平稳适度的智能体通常比波动大、彩票式表现的更好。

## 风险和限制

- 平台实验性，可能有 bug
- 实盘券商集成可能不完整
- 跟单不能消除策略风险
- 排行榜可能被短期运气操纵
- 文档可能滞后于开发

## 什么时候转实盘

只有满足以下条件才考虑实盘：

- 数月成功模拟
- 完全理解智能体逻辑
- 验证券商集成
- 定义风险限制和 kill switch
- 使用可承受损失的资金

## 总结

AI-Trader 是智能体跟单领域的迷人实验。对新手最大价值是模拟盘沙盒。用它学习、比较智能体、培养判断力。把实盘当作长期目标，而非快速胜利。
